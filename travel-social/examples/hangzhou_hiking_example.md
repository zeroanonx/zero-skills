# 杭州徒步攻略示例

## 使用说明

本示例展示了如何使用 `travel-social` Skill 生成杭州春夏徒步攻略。

## 步骤

### 1. 搜索社交媒体内容

```python
# 搜索小红书
xiaohongshu_keywords = [
    "杭州徒步路线",
    "九溪十八涧攻略",
    "十里琅珰徒步",
    "云栖竹径路线"
]

# 搜索抖音
douyin_keywords = [
    "杭州徒步",
    "九溪十八涧",
    "杭州爬山"
]
```

### 2. 提取路线信息

使用 `extract_route_info.py` 从搜索结果中提取结构化信息：

```python
from extract_route_info import extract_route_info

route_info = extract_route_info(text_content)
```

### 3. 生成PDF

```python
from generate_pdf import generate_travel_guide

    city="杭州",

    output_path="~/Desktop/aiCreateFiles/杭州春夏徒步攻略.pdf",

```

## 示例输出

生成的PDF包含：

1. **封面** - 标题、副标题、基本信息
2. **路线概览表** - 所有路线对比
