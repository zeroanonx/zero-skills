# 时间轴旅游攻略示例

本示例展示如何使用 `travel-social` Skill 生成包含详细时间轴的旅游攻略。

## 适用场景

- 跨城市旅行（杭州→南京、上海→苏州等）
- 多日行程规划（2-3天）
- 需要精确时间安排的旅行
- 美食+景点结合的深度游

## 生成的PDF内容结构

### 1. 封面
- 标题：杭州 ➡️ 南京 两天一夜深度游
- 副标题：详细时间轴 | 交通美食全攻略
- 基本信息：出发地、目的地、天数、出发时间

### 2. 行程总览
- 日期：12.07(周日) - 12.08(周一)
- 完整路线：南京南 ➡️ 总统府 ➡️ ... ➡️ 大屠杀纪念馆
- 需提前预约清单（5个景点）

### 3. 时间轴表格（核心）

**Day 1 示例**：

| 时间 | 游玩安排 | 交通细节 | 小提示 | 美食 |

| 10:30<br/>～<br/>11:50 | **总统府**<br/>80分钟 |  | 提前在公众号预约<br/>08:30-17:00开放 |  |

| ... | ... | ... | ... | ... |

**特点**：
- ✅ 精确到分钟的时间安排
- ✅ 优先打车方案（含价格）
- ✅ 每个景点的小贴士
- ✅ 对应时间段的美食推荐

### 4. 酒店推荐
- 如家酒店neo(南京夫子庙店)

## 数据结构

```python
@dataclass
class TimelineItem:
    time_start: str      # "10:00"
    time_end: str        # "11:50"
    activity: str        # "总统府"
    activity_type: str   # "景点"
    duration: int        # 80 (分钟)
    transport: str       # "打车18分钟/12元"
    transport_detail: str
    tips: str            # "提前在公众号预约"
    food: str            # "鸭血粉丝汤 15-25元"
    booking_info: str    # "需预约"
```

## 优势

1. **时间精确**：每个活动都有明确的起止时间
2. **交通详细**：优先打车，包含时长和预估费用
3. **美食结合**：每个时间段都有对应的餐饮推荐
4. **实用性强**：包含预约提醒、拍照点、注意事项
5. **一目了然**：表格形式，方便查看和打印
