# Skill: travel-social

## 概述

从社交媒体平台（小红书、抖音等）提取真实用户分享的旅游信息，生成详细、实用的旅游攻略PDF。

## 适用场景

- 需要获取真实、实时的旅游信息（非官方宣传）
- 想要避坑指南和实用建议
- 需要包含最新路况、景点开放情况
- 想要发现小众、网红打卡点

## 支持的数据源

| 平台 | 内容类型 | 优势 |
|------|---------|------|
| 小红书 | 图文笔记 | 详细攻略、实拍照片、避坑经验 |
| 抖音 | 短视频 | 直观展示、实时路况、路线演示 |

## 工作流程

```
1. 搜索社交媒体内容
   └── 关键词搜索小红书、抖音
   └── 提取高赞、高收藏内容

2. 信息提取与整合
   └── 路线信息（起点、终点、途经点）
   └── 时间节点（各段耗时）
   └── 实用信息（交通、餐饮、装备）
   └── 避坑建议（评论区高赞）

3. 生成攻略PDF
   └── 精美排版
   └── 结构化内容
   └── 可打印格式
```

## 输出格式

### 1. 标准路线格式
适合徒步、骑行等单条路线攻略，包含：
- 路线概览（距离、时长、难度）
- 分段详细说明
- 景点亮点
- 交通、装备、美食建议

### 2. 时间轴行程格式（NEW）
适合多城市、多天的详细行程规划，包含：
- **精确时间节点**：每个景点的到达/离开时间
- **详细交通方案**：优先打车，包含时长和价格
- **美食推荐**：每个时间段对应的餐厅/小吃
- **游玩要点**：景点特色、拍照点、注意事项
- **预约提醒**：需要提前预约的景点清单
- **携带清单**：必备物品

**示例场景**：
- 杭州到南京两天游
- 上海周末 citywalk
- 成都美食三日游

## 使用方法

### 基本用法

生成某城市旅游攻略：

```
用户：帮我生成杭州徒步攻略
系统：
1. 搜索小红书"杭州徒步"、"九溪十八涧"等关键词
2. 搜索抖音相关视频
3. 提取路线、景点、美食信息
4. 生成详细PDF攻略
```

### 进阶用法

指定主题和季节：

```
用户：生成成都美食攻略，侧重火锅和小吃
系统：搜索成都火锅、小吃相关内容，生成美食专题PDF
```

指定难度和时间：

```
用户：北京周边一日游，轻松休闲路线
系统：筛选轻度徒步/休闲路线，生成一日游攻略
```

### 时间轴行程格式（详细版）

适合需要精确时间规划的旅行：

```
用户：杭州到南京两天旅游攻略，从早上10点出发
系统：
1. 规划详细时间节点（10:00出发、10:30到达总统府等）
2. 安排交通方式（优先打车，包含时长和费用）
3. 推荐每个时间段的美食
4. 列出需要预约的景点
5. 生成包含时间轴表格的PDF
```

**时间轴格式包含**：
| 列 | 内容 |
|---|---|
| 时间 | 精确到分钟的起止时间 |
| 游玩安排 | 景点名称 + 游玩时长 |
| 交通细节 | 打车/地铁方案、时长、价格 |
| 小提示 | 预约信息、拍照点、注意事项 |
| 美食 | 推荐餐厅/小吃 + 价格 |

**示例输出**：
- Day 1: 总统府(10:30-11:50) ➡️ 南京博物院(12:50-15:30) ➡️ 秦淮河夜游
- Day 2: 鸡鸣寺(10:30-11:30) ➡️ 城墙(12:00-13:00) ➡️ 纪念馆

## 输出格式

PDF包含以下章节：

1. **封面**
   - 标题、副标题
   - 适用人群、难度等级
   - 最佳季节、路线概览

2. **第一章：目的地概览**
   - 为什么选择这里
   - 最佳旅游时间
   - 天气与穿着建议

3. **第二章：精选路线详解**
   - 每条路线的概览信息
   - 详细分段说明
   - 景点亮点
   - 实用贴士

4. **第三章：实用信息**
   - 交通指南（表格）
   - 装备清单
   - 餐饮推荐
   - 安全注意事项

5. **第四章：行程规划建议**
   - 单人出行建议
   - 结伴出行建议
   - 四季特色推荐

6. **结语**

## 信息提取规则

### 路线信息提取

从社交媒体内容中提取：
- 路线名称
- 起点/终点
- 途经景点
- 全程距离
- 预计耗时
- 难度等级

### 实用信息提取

- **交通**：地铁线路、公交站点、自驾停车
- **餐饮**：沿途餐厅、特色小吃、人均消费
- **装备**：必备物品、季节专属装备
- **避坑**：评论区高赞的注意事项

### 信息可信度评估

- ✅ 高赞（>1000赞）+ 近期发布（<3个月）
- ✅ 多来源交叉验证
- ⚠️ 单一来源或过时信息（>6个月）
- ❌ 明显广告或商业推广

## 实际使用说明

### 关于数据源访问限制

**重要提示：** 由于小红书和抖音的内容访问限制，无法直接通过websearch获取详细的用户笔记内容。这是这些平台的反爬机制导致的。

### 替代方案

当无法直接获取社交媒体内容时，可采用以下策略：

#### 方案1：混合数据源（推荐）
结合多个信息源：
- **携程攻略** (you.ctrip.com) - 详细的路线攻略
- **Trip.com Moments** - 用户分享的游记
- **网易、搜狐旅游** - 综合旅游文章
- **YouTube** - 视频攻略

#### 方案2：人工收集 + 自动整理
1. 手动在小红书/抖音搜索并浏览内容
2. 复制关键信息（路线、时间、贴士）
3. 使用本Skill的`extract_route_info.py`提取结构化数据
4. 使用`generate_pdf.py`生成攻略

#### 方案3：使用聚合平台
- **马蜂窝** (mafengwo.cn) - 有大量用户游记
- **穷游** (qyer.com) - 详细的旅行攻略
- **大众点评** (dianping.com) - 餐饮和景点信息

### 实际案例：杭州春夏徒步攻略

本项目已成功制作了**杭州春夏徒步攻略PDF**（见`examples/`），使用的是**方案1（混合数据源）**：

**使用的信息源：**
- Trip.com用户游记（香港用户分享的杭州徒步路线）
- 杭州网官方攻略（五大山峰徒步路线）
- 每日商报（三条人少景美的踏春路线）
- 杭州市政府官网（西湖景区官方信息）
- 网易旅游（九溪十八涧详细攻略）
- YouTube视频（西湖群山标毅线）

**生成的PDF包含：**
- 4条精选路线（6-8公里，2-4小时）
- 详细的路线分段说明
- 交通、装备、餐饮信息
- 安全注意事项
- 单人和结伴出行建议
